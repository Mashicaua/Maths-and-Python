# 2. if statement
## Python
```Python
if x == 5l:
    print(x)
```
## Math
$x;$ if $x = 51$
## Explanation
We can use ";" symbol to represent the end of an expression.  
It is very important to write keywords and functions not in italics or cursive.  
### Structure
$$\LARGE{[expression];\textsf{ if } [condition]}$$
